PHP Database Migrations for Everyone

Phinx makes it ridiculousy easy to manage the database migrations for your PHP app. In less than 5 minutes you can install Phinx using PEAR and create your first database migration. Phinx is just about migrations without all the bloat of a database ORM system or dependence on an application framework.

Note: Phinx is currently under heavy development and is therefore super-alpha software.

Installing

By far the best way to install Phinx is via PEAR:

$ sudo pear channel-discover pear.phinx.org
$ sudo pear install channel://pear.phinx.org/Phinx-1.1.tgz

Phinx will now be available under the phinx command.

Writing Your First Migration

$ cd yourapp
$ phinx init .
$ vim phinx.yml
$ mkdir migrations
$ phinx create MyFirstMigration
$ phinx migrate -e development

And that's it. Phinx will connect to your development database and execute your first migration.
